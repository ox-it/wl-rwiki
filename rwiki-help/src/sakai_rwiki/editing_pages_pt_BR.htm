Editar suas páginas

Para editar o conteúdo de uma página, selecione "Editar" botão na parte superior da página. Faça suas alterações na caixa de texto. Em seguida, selecione o botão 'Salvar' no topo da página.   Suas alterações aparecerão na página wiki para todo mundo ver.

Se você receber uma mensagem Permissão Negada quando você tentar editar uma página, você não tem permissão para atualizar páginas. Se você acha que isso é um erro, você terá que falar com o administrador do site para mudar isso.

Formatar sua página

Quando você edita a página, você verá uma "Imitação da Folha" exibido na página à direita. Isso lhe dá uma breve introdução aos códigos utilizados para formatação de texto na Wiki. Use o "Imitação da Folha" para uma página de ajuda completa. Use esta página de ajuda para obter informações completas sobre a formatação do texto, incluindo expoentes, diferentes tipos de listas, etc

Link para outra página wiki

Para adicionar um link para outra página Wiki, digite o nome da página entre colchetes. Por exemplo, digitando "[página 2]" irá vincular a uma página chamada "página 2".

Se você excluir acidentalmente algo que você não queria, isso não é um problema. Veja o histórico da página para obter mais detalhes.

Link para uma página web externa

Para adicionar um link para uma página web externa, digite a URL completa, incluindo o 'http://'. Por exemplo, 'http://www.bbc.co.uk'.

Você também pode clicar no 'link externo', que irá inserir um exemplo de link que você pode editar.

O que acontece se alguém estiver alterando a página ao mesmo tempo?

Se alguém editou e salvou a página que você estava editando, você verá uma mensagem que diz que a página foi alterada e sua nova página será exibida acima. Você pode então decidir por substituir suas alterações (utilizando o botão 'Substituir') ou para manter suas alterações (utilizando o botão 'salvar').
