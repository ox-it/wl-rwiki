Visualizar o histórico da página

Para obter o histórico de qualquer página, selecione a opção 'histórico' no menu.

Isso dá a você as seguintes opções

  1. Fornece uma lista de todas as versões anteriores dessa página, com o nome do usuário que alterou e a data.
  2. Permite que você visualize qualquer versão anterior.
  3. Compare o conteúdo de qualquer versão com a versão atual.
  4. Compare o conteúdo de qualquer versão com a versão imediatamente anterior, para ver quais alterações foram feitas por cada usuário.
  5. Permite que você reverta para qualquer versão anterior.

Se você apagou acidentalmente o conteúdo de sua página, você pode usar a opção "reverter para esta versão" para reverter para uma versão anterior e correta, por exemplo.
