Visualizar as informações da página

Para obter as informações de qualquer página, selecione a opção 'info' no menu.

Isso lhe dá a seguinte informação

  1. Listas de todas as páginas da página de links para
  2. Lista todas as páginas que apontam para esta página
  3. Fornece a página do proprietário (geralmente a pessoa que o criou)
  4. Fornece o nome da página global, para permitir links para ele de fora da Wiki
  5. Fornece os detalhes da página de permissões (o site que você precisa ser um membro de vê-lo)
  6. Dá detalhes de quando a página foi editada

'Info' também permite que você abra as seguintes versões da página:

  1. Versão para impressão
  2. Não editável versão HTML
  3. Uma versão do formato de documento aberto (.odf) (que pode ser aberto no Open Office, por exemplo)
  4. Um feed RSS de alterações recentes
