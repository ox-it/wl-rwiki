Criar uma nova página wiki

Para criar uma nova página, crie um link para ela de uma página existente. Por exemplo, se você ainda não tem uma página chamada "página 2", escrevendo "[página 2]" irá criar uma nova página vazia chamada "página 2".

Você não pode usar os seguintes caracteres em um link ou título da Wiki:

  • : Dois pontos (isso pode mudar)
  • @ Arroba
  • # Hash
  • | Barra Vertical
  • \ Barra Invertida
