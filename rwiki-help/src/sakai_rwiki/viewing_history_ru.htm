Просмотр истории изменений

Чтобы просмотреть историю изменений любой страницы Вики, нажмите на ссылку "История" вверху страницы.

Это позволит вам:

  1. Просмотреть список предыдущих версий страницы, имя пользователя, который вносил изменения, и дату изменения.
  2. Просмотреть любую из предыдущих версий.
  3. Сравнить содержимое любой из предыдущих версий с последней версией документа.
  4. Сравнить содержимое любой из предыдущих версий с той, которая ей непосредственно предшествовала, чтобы увидеть какие изменения вносил каждый из пользователей.
  5. Восстановить любую из предыдущих версий.

Например, если вы случайно удалили со страницы все содержимое, то вы можете восстановить предыдущую версию, на которой содержимое еще не удалено.
